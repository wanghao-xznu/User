/*******************************************************************************
** 实验说明：
   1、除了常规的配置外，还要配置USART1模块(115200baud，8位数据位，1位停止位，无奇偶校验位)

   2、本例程通过串口实现了人机交互，MCU打印字符，提示用户输入，并且将此字符输出
      注意字符的ASIC值在0~255之间，多余的会被丢弃。

** 实验现象：
   1、打开超级终端，对串口要设置相同的参数(可以参考我们提供的资料)
   2、通过JLINK下载后，将程序烧写进MCU，按下“RESET”键，超级终端接收到了字符串
	“ Welcome to www.OPENMCU.com
	   Please Input Character From Keyboard ”
      我们键入一个字符，PC会将我们输入的字符打印出来。
  *****************************************************************************/
